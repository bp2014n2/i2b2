Demographics (1 Patient Set) - Simple Counts

Introduction

This plugin displays demographic information for a patient set across several different categories.

Instructions

Navigate to the "Specify Data" tab. Then, drag and drop a Patient Set onto the input box. Finally, click the "View Results" tab to view the demographic details.
