General usage information

This is general information about the "report: Statistical functions" webclient plugin. It is intended for end users - as scriptlet developer please refer to the "Notes for scriptlet developers". For information about a specific scriptlet please read the description notes or contact the scriptlet developer.

Using this webclient plugin is simple: At first choose the desired scriptlet from the dropdown list. Then read the scriptlet instructions for specifying patient sets and concepts by drag and drop. Theoretically you can drag and drop an unlimited number of patient sets and concepts on the appropriate fields. But in practice the actually used fields by the scriptlet are limited - so just follow the instructions. Also note that for every patient set there will be one request to the CRC cell on server side - this will take some time and could cause a timeout if server is busy (or runs in a low performance environment like a virtual machine).

Finally specify additional input values (if available) and click on "View results".

Known issue: Older web browsers can't handle the .svg format, that the plots have. So plots won't be displayed in this case. This is also true for newer browsers running in an old compatible mode e.g. IE9 in IE7 mode. As simple solution just click on the button below to download the file and use another program to view it.
